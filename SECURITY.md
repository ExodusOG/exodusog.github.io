# Security Policy

## Reporting a Vulnerability

If you believe you have found a security vulnerability related to this repository
or any content published at exodus-data.com, please report it responsibly.

**Email:** contact@exodus-data.com

Please include, if possible:
- A clear description of the issue
- Steps to reproduce the behavior
- URLs, files, or paths involved
- Any relevant logs or screenshots

Do **not** publicly disclose security issues without prior coordination.

---

## Scope

This repository hosts static, publicly accessible content.
There is no user authentication, data collection, or backend processing.

Only issues that directly affect the integrity, availability, or correctness
of the published content are considered in scope.

---

## Response Expectations

Reports are reviewed on a best-effort basis.
If an issue is confirmed, reasonable steps will be taken to address it.

No guaranteed response time is provided.
