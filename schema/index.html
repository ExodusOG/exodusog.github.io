Conventions

  • Stable URLs: section pages use /section/ and serve index.html.
  • Dataset IDs: stable identifiers like REG_US_CFR_001 or LIC_OS_001.
  • Dates: ISO 8601 (YYYY-MM-DD).
  • Machine-readable: include a JSON metadata block on dataset pages.
  • Sources: include source name + URL(s) for factual datasets.
