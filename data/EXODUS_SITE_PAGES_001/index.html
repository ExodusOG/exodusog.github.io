Summary

This dataset is a stable, machine-readable index of Exodus Data’s public pages, including licensing entry points and the authoritative machine-readable license text.

Crawling and indexing are permitted under the site license, but commercial use requires a paid commercial license. See /license/ (authoritative text: /LICENSE.txt).

Download

JSON: /data/EXODUS_SITE_PAGES_001/dataset.json

This dataset is intentionally small and stable. It establishes the format for future datasets.

Fields

  • id — dataset identifier
  • published_at, updated_at — ISO 8601 date
  • license — license policy URL
  • machine_license — authoritative machine-readable license URL
  • pages[] — list of canonical site pages with path, url, type, and notes
